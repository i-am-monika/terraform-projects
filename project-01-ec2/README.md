# Project 1 — EC2 Instance with Terraform

Provisioned an AWS EC2 t2.micro instance using Terraform demonstrating the core IaC workflow.

## What I Built
- Launched an EC2 instance in ap-south-1 using Terraform
- Used variables for reusable configuration
- Captured outputs — instance ID, public IP, state

## Tech Stack
Terraform · AWS EC2 · HCL

## Commands Used
terraform init → terraform plan → terraform apply → terraform destroy
